# M-Westy Outfits

Uma solução de vestiário minimalista e intuitiva para FiveM, desenvolvida com foco em alta performance e refinamento visual.

---

### Especificações

* **Compatibilidade** · Suporte nativo para vRPex e Creative.
* **Performance** · Estrutura leve e otimizada (baixo consumo no resmon).
* **Instalação** · Configuração simplificada e adaptável.

### Funcionalidades

* **Customização** · Sistema de presets totalmente personalizável.
* **Interface** · UI moderna, limpa e rica em detalhes.
* **Gerenciamento** · Ajustes através do arquivo de configuração.

---

### Distribuição & Proteção

Este recurso é gratuito. Para salvaguardar os direitos autorais do projeto e evitar a cópia não autorizada da interface ou revendas fraudulentas, a estrutura lógica (`client` e `server`) e a interface visual (`front-end/html`) encontram-se criptografadas. 

Apenas o arquivo de configurações (`config.lua`) e o manifesto permanecem abertos, garantindo total autonomia para você personalizar os presets do seu servidor.

**Termos de Uso**  
Protegido pela Lei do Software (Nº 9.609/98) e Art. 184 do Código Penal. É proibida a redistribuição indevida ou revenda deste código fora dos canais oficiais.
